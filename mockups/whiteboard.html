MINIMUM HOME BENCH

  1. M5Stack controller + local status display.
  2. One motion/light sensor and one quiet actuator.
  3. Local sampling, timestamps, buffering, health checks.
  4. Then add inward-facing camera and optical enclosure.

DATA PATH

physical source + sensors + camera → sample / timestamp / health checks → frame commitment + difference metrics → mix with device CSPRNG → signed observation envelope → outbound HTTPS → Access-protected Worker → Durable Object (sequence + replay + live state) → Queue / Workflow → D1 metadata + sampled R2 evidence

TRAFFIC, HONESTLY

  • Aggregate, approved traffic may drive light or actuator rhythm.
  • It is public context / an additional mixed input.
  • It is not treated as secret physical entropy.

FAIL-SAFE TESTS

  • Frozen/repeated frame
  • Covered camera
  • Stopped actuator
  • Implausibly stable sensor
  • Reboot and sequence reset
  • Network loss and buffered recovery
